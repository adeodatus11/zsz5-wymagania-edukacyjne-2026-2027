# Przebudowa wymagań ogólnokształcących ZSZ5 2026/2027

Data utworzenia: 2026-06-09.

Ten folder zbiera pliki robocze do naprawy strony:

- `../wymagania_ogolne_ZSZ5_2026_2027.html`

Punkt wyjścia: audyt wykazał, że obecny HTML zawiera dużo treści z podstaw programowych, ale nie jest jeszcze wiarygodnym dokumentem wymagań edukacyjnych na oceny. Największy problem nie polega tylko na brakujących działach, lecz na tym, że wymagania zostały mechanicznie rozłożone po kolumnach ocen.

## Pliki robocze

1. `model_wymagan_na_oceny_ZSZ5.md`
   - wspólny model ocen `dopuszczająca` - `celująca`;
   - zasady dla przedmiotów teoretycznych, praktycznych, WF, edukacji zdrowotnej i edukacji obywatelskiej.

2. `macierz_zrodel_wymagan_ogolnych_ZSZ5.md`
   - lista przedmiotów ze strony HTML;
   - lokalne źródła PDF;
   - status kompletności i ryzyka.

3. `lista_brakow_i_kolejnosc_przebudowy.md`
   - konkretne braki z audytu;
   - kolejność naprawy;
   - decyzje, czego nie publikować przed zatwierdzeniem.

4. `pakiet_dla_claude.md`
   - krótki kontekst do przeniesienia pracy do Claude;
   - gotowe polecenia do dalszego uzupełniania przedmiotów.

5. `edb_technikum_pilot_przebudowy.md`
   - pierwszy wzorcowy przedmiot po audycie;
   - zawiera brakujące działy II i III;
   - pokazuje docelowe stopniowanie wymagań na oceny.

6. `plan_unifikacji_stron_ogolne_zawodowe.md`
   - plan ujednolicenia strony ogólnokształcącej i zawodowej;
   - wskazuje elementy wspólne bez mieszania różnych struktur danych.

## Decyzja audytowa

Obecny HTML może zostać jako interfejs i surowy indeks treści, ale finalne wymagania edukacyjne powinny powstać dopiero po:

1. potwierdzeniu źródła podstawy programowej dla każdego przedmiotu;
2. uzupełnieniu wszystkich działów;
3. przepisaniu wymagań na jeden model ocen;
4. ręcznej recenzji nauczyciela przedmiotu.
